# REQUEST_CPP

Simple request library in c++

## Installation
Add the files (request_cpp.h, request_cpp.cpp) to your project
```cpp
#include <iostream>
#include "request_cpp.h"
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
